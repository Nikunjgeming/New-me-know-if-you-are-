# QuickSMS - Premium Virtual Numbers

A premium web interface for QuickNum API to get virtual numbers for SMS verification.

## Features

- 🎨 Modern, premium UI design
- 📱 Get virtual numbers for any service
- 🔍 Search services easily
- 📩 Request another SMS on same number
- 📊 View activation status
- 🔄 Auto-refresh status
- 📋 Copy number with one click
- 📜 Activation history

## Technologies Used

- HTML5
- CSS3 (Custom)
- JavaScript (Vanilla)
- QuickNum API
- Font Awesome Icons
